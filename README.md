# VisualForge AI v10

AI background + uploaded Canva/product image + real canvas overlay + export.

## What this build does

- Pollinations free background generation
- No Puter script and no Puter balance popup
- Platform size presets: Etsy, Facebook, Pinterest, LinkedIn, Blog, Website, Roofing
- Upload Canva/product image(s)
- Product placement modes: right card, left card, center card, bottom stack, none
- Overlay templates: Etsy, Facebook, Story, Pinterest, Website, Roofing, Business, Banner
- Custom readable overlay text
- Custom colors with color pickers
- Product scale, X/Y position, rotation
- PNG/JPG/WebP download
- ZIP download for all generated final images

## GitHub Pages install

Upload these files to the repo root:

- `index.html`
- `app-v10.js`
- `style-v10.css`
- `README.md`

Then open:

`https://YOUR-USERNAME.github.io/YOUR-REPO/?v=10`

## Best prompt rule

Write prompts for the background only. Do not ask AI to create readable text.

Good prompt:

`cozy beige desk flat lay, clean printable product mockup background, soft natural light, empty space for overlay, realistic product photography, no text, no logo`

Bad prompt:

`make an Etsy image with title Weekly Planner and badges Editable in Canva`

The overlay builder adds real readable text after generation.
